# SMM-unicsul
Simulador de meio de pagamentos em python para treino de lógica e organização de código

## Objetivo
Aplicar conhecimentos de algoritmos matemáticos e pensamento computacional desenvolvendo um simulador de meio de pagamento (SMM) em Python. Para as operações de debito é calculado 1% de MDR (valor_taxa da transaçao), e para cartao de credito e credito parcelado é calculado 5% de MDR.

## Como rodar o projeto

#### 1 Instalar o Python

Tenha o python instalado para poder executar o arquivo `main.py`

#### 2 Clonar o repositório e acessar o local

```
git clone https://github.com/anselmosz/smm-unicsul

cd <path to>/smm-unicsul
```

#### 3 Execute o arquivo `main.py` através da linha de comando de sua preferência

```
python main.py
```
